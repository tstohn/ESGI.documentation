# Application


## Multimodal

Application example for multimodal data using SIGNALseq, containing both transcripts and protein levels, generated by combinatorial indexing. 

Download the HELA data of SIGNAL-seq:

```
---
wget "https://sra-pub-run-odp.s3.amazonaws.com/sra/SRR28056729/SRR28056729" -O SRR28056729.sra
wget "https://sra-pub-run-odp.s3.amazonaws.com/sra/SRR28056728/SRR28056728" -O SRR28056728.sra
fastq-dump --split-files --gzip SRR28056729.sra
fastq-dump --split-files --gzip SRR28056728.sra

# Or download fastqfiles directly from ENA (European Nucleotide archive)
wget "https://ftp.sra.ebi.ac.uk/vol1/fastq/SRR280/029/SRR28056729/SRR28056729_1.fastq.gz"
wget "https://ftp.sra.ebi.ac.uk/vol1/fastq/SRR280/029/SRR28056729/SRR28056729_2.fastq.gz"
wget "https://ftp.sra.ebi.ac.uk/vol1/fastq/SRR280/028/SRR28056728/SRR28056728_1.fastq.gz"
wget "https://ftp.sra.ebi.ac.uk/vol1/fastq/SRR280/028/SRR28056728/SRR28056728_2.fastq.gz"
---
```
